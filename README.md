E-Commerce Revenue Analysis

This project is a Tableau dashboard created with an e-commerce retail sales dataset from Kaggle.

The dashboard shows:

* sales and revenue analysis
* product category performance
* monthly sales trends
* customer purchasing behavior

Dataset

Dataset source:
[Retail Sales Dataset - Kaggle](https://www.kaggle.com/datasets/luisrenterialezano/retail-sales-dataset?)

Tools Used

* Tableau Public
* Data Visualization
* SQL (DuckDB)
